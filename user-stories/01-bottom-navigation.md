# Bottom Navigation

## Summary
Mobile-friendly bottom tab bar for navigating between the three core sections of the app.

## Tabs
- **Meal Planner** (`/meal-planner`) — weekly meal schedule
- **Recipes** (`/recipes`) — search, create, and AI-generate recipes
- **Grocery List** (`/groceries`) — current shopping list (existing feature)

## Behaviour
- Always visible at the bottom of the viewport on all pages.
- Active tab is highlighted.
- Uses TanStack Router for URL-based navigation.
- PWA standalone mode: respects safe-area insets.
